<div class="statute-body">
  <p class="block subsection">(1) The Authority shall enable an Aadhaar number holder to lock his/her Aadhaar number and unlock it when needed for authentication.</p>
  <p class="block subsection">(2) All authentication requests using Aadhaar number against any such locked Aadhaar number shall result with a “No” answer with an appropriate response code.</p>
  <p class="block subsection">(3) In case of a locked Aadhaar, the Authority will allow the resident to authenticate using Virtual ID or other means.
